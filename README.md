# Gesture-Controlled 3D Particle Simulation System

## Overview
This project implements an interactive 3D particle system using real-time rendering and gesture-based interaction concepts.

## Highlights
- 3D particle system with 8000 particles  
- Real-time animation and rotation  
- Mathematical distribution of particles in space  
- Designed for integration with gesture-based control  

## Methodology
- Particle rendering using Three.js  
- BufferGeometry for efficient particle handling  
- Real-time animation loop using requestAnimationFrame  

## Tech Stack
- JavaScript  
- Three.js  

## Applications
- Interactive visualization  
- Simulation systems  
- Computer graphics experiments
